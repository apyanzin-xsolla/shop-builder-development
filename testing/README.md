# AI Kit Testing Artifacts

Current canonical artifacts:

- `ai-kit-operational-metrics.md` — measurable A/B success metrics and formulas.
- `ai-kit-skill-eval-matrix.md` — per-skill prompts, required checks, targets, and failure conditions.
- `ai-kit-eval-cases.yaml` — production-like eval cases for AI Kit vs no-AI-Kit comparison.
- `ai-kit-ab-pilot-results.md` — pilot A/B result for 4 ready skills.
- `ai-kit-production-informed-results.md` — production-informed 3-variant comparison using Slack-derived real-risk cases.
- `ai-kit-eval-results.md` — short result summary and launch-quality rerun guidance.
- `sandbox-runbook.md` — how to run live sandbox/prod-like validation safely when credentials are available.
- `scripts/ai_kit_eval.py` — reusable all-in-one eval script for future skills: score, report, dashboard data, and canvas.
- `scripts/score_ai_kit_eval.py` — no-dependency scorer for AI Kit vs baseline eval runs.
- `ai-kit-eval-runs.sample.json` — sample input data for the scorer.
- `ai-kit-eval-score.sample.json` — sample scored output generated by the script.
- `ai-kit-real-eval-runs.json` — real transcript-derived eval run metrics.
- `ai-kit-real-eval-score.json` — scored output from real transcript-derived metrics.
